## Execution des requêtes
Depuis le dossier traitement_requets, executer le script shell.py

```
python shell.py -m <b|v|v2> [-t]
```

#### Arguments : 
-m --model      : research model chosen for the search. ['b','boolean', 'v', 'vector']
-t              : enable the time record.
